1. ACA_I.m 						(Almost Correct Adder)
2. ETAII.m 						(Error Tolerant Adder Type II)
3. ACA_II.m						(Accuracy Configurable Adder)
4. GDA.m						(Gracefully Degrading Adder)
5. GeAR.m 						(Generic Accuracy Configurable Adder)
6. Error_Probability.m 			(Calculates the Probability of Error for any GeAr adder)
7. Carry_Generate.m 			(Internal Function for carry generation)
8. Error_Probability_Internal.m (Internal Function for error probability)
